Тестовое задание для Аякс. При вводе в инпут через debounce отправляется запрос постов к api и полученные посты выводятся списком элементов, которые кликом можно развернуть для более подробной информации

Для запуска через docker:

    - docker build -t task .
    - docker run -p 3005:3005 task

Для запуска проекта через npm:

    - npm i
    - npm run dev

Т.к. api оказался недоуступен в России, запросы идут через написанный прокси, иногда некоторые запросы могут грузиться несколько секунд. Сам api: https://dummyjson.com/
